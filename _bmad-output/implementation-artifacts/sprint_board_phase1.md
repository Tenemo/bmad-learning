# Sprint board phase 1

## Sprint goal

Deliver a complete, tested, and containerized single-user todo application with BMAD artifacts and QA evidence.

## Story order

1. Story 1.1: scaffold the workspace and test harness
2. Story 1.2: produce the BMAD planning artifacts
3. Story 2.1: initialize SQLite persistence and health checks
4. Story 2.2: implement todo CRUD endpoints with validation
5. Story 2.3: cover backend behavior with integration tests
6. Story 3.1: build the core layout and list rendering
7. Story 3.2: implement optimistic create, complete, and delete flows
8. Story 3.3: implement loading, error, and accessibility behavior
9. Story 3.4: cover frontend behavior with component tests
10. Story 4.1: build the Playwright journey suite
11. Story 4.2: containerize the application with Docker Compose
12. Story 4.3: generate QA reports and finalize documentation

## Definition of done

- implementation complete
- tests added or updated
- lint and type-check passing
- story notes updated
- review complete
